If the `codemd` MCP server is connected, prefer its tools (`codemd_search_artifacts`, `codemd_get_impact_radius`, `codemd_get_callers`, `codemd_get_callees`, `codemd_get_call_paths`, `codemd_read_artifact`) before CodeGraph or raw search/read when exploring code, tracing call paths, estimating impact, or editing code. Start with `codemd_search_artifacts` to find likely symbols, then use the callgraph tools as appropriate. Verify against actual source before finalizing a change; the underlying callgraph can be stale or regex-inferred.

These `codemd_*` tools are frequently deferred (not loaded as directly callable) in Claude Code sessions with many active tools/MCP servers — check the deferred-tools system-reminder, and if `codemd_*` names appear there, call `ToolSearch` with `select:codemd_search_artifacts,codemd_get_impact_radius,codemd_get_callers,codemd_get_callees,codemd_get_call_paths,codemd_read_artifact,codemd_semantic_search,codemd_find_tests,codemd_review_changes` to load their schemas before the first use in a session, rather than falling back to grep/read.
